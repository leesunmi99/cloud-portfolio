## RollingUpdate
- 배포 작업: Deployment 업데이트
- 특징: 자동 배포(정지, 롤백 가능) / 트래픽 제어 불가 / 무중단 서비스 지원
- 단점: v1, v2 동시호출 발생

## Blue/Green
- 배포 작업
  1) v2 Deployment 생성
  2) Service의 Selector 변경
  3) v1 Deployment 삭제
